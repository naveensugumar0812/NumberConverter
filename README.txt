TECHNOLOGY
----------
STS IDE 4

SPRING BOOT framework

RUN Application:
----------------
* Download application from github : https://github.com/naveensugumar0812/NumberConverter
* Open file NumberWordConverterApplication.java ,right click Run application as Java Application or Spring boot app
* once server is started hit the url "http://localhost:8090/converter/<number to be tested>". Eg: http://localhost:8090/converter/9999.
